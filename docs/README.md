# Territórios App

Sistema para gerenciamento de territórios de congregações das Testemunhas de Jeová.

## Tecnologias

- Next.js 16
- TypeScript
- Supabase
- PostgreSQL
- Vercel

## Objetivo

Permitir que cada congregação administre seus territórios, endereços, designações e mapas de forma simples e segura.

## Princípios

- Segurança em primeiro lugar
- Cada congregação acessa apenas seus dados
- Código organizado e reutilizável
- Interface simples para os irmãos
